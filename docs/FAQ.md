# 常见问题 (FAQ)

## 安装和启动

### Q: 初始化需要多久？
下载约 2.3GB 文件，取决于网络速度，通常 5-15 分钟。

### Q: macOS 提示"无法打开"怎么办？
右键点击 `.command` 文件，选择"打开"，然后在弹窗中点击"打开"。或在"系统设置 > 隐私与安全性"中允许运行。

### Q: Windows 提示"Windows 保护了你的电脑"怎么办？
点击"更多信息"，然后点击"仍要运行"。

### Q: 需要安装什么软件吗？
不需要。PocketClaw 自带 Node.js 运行时，双击启动脚本即可。

## 使用

### Q: 支持哪些 AI 模型？
DeepSeek、Kimi (月之暗面)、通义千问、智谱 GLM、Claude (Anthropic)、GPT (OpenAI) 等。

### Q: API Key 从哪里获取？
从各模型提供商官网获取。推荐 DeepSeek (https://platform.deepseek.com)，注册即可获得免费额度。

### Q: 可以离线使用吗？
不可以。AI 对话需要调用云端模型 API，需要网络连接。未来可能支持本地模型 (Ollama)。

### Q: U 盘换电脑还能用吗？
可以。所有数据存在 U 盘中，换到任何 Mac 或 Windows 电脑都能使用。

### Q: 聊天记录存在哪里？
存在 U 盘的 `data/` 目录中。

## 故障排除

### Q: 启动后浏览器空白？
等待几秒让 Gateway 启动完成，然后刷新页面。

### Q: 提示"Gateway 未启动"？
检查端口 18789 是否被占用。可以在管理菜单中重新启动。

### Q: 聊天没有回复？
1. 检查网络连接
2. 确认 API Key 正确
3. 确认模型提供商服务正常
4. 查看 Gateway 日志排查错误

## 安全

### Q: 我的 API Key 安全吗？
API Key 存储在 U 盘的配置文件中。请：
1. 妥善保管 U 盘
2. 在模型提供商处设置使用限额
3. 不在不信任的电脑上使用

### Q: 这个软件会收集我的数据吗？
不会。PocketClaw 完全开源，所有数据存在本地 U 盘中，不会上传到任何服务器。
